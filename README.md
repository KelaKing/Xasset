# Xasset

Export images from xcassets and replace images to xcassets.

## Installing
### [Mint](https://github.com/yonaskolb/mint)
```
$ mint install KelaKing/Xasset
```
## Usage
```sh
# Export images
xasset export ~/Foo.xcassets ~/FooImages
# Replace images
xasset replace ~/FooImages ~/Foo.xcassets
```
